How to Use This Website

Learn how to navigate the YAA.Group website, find information, and use accessibility and language features.

Navigation Guide

Our website is organized to help you quickly find the information you need about YAA.Group's platforms, services, and resilience solutions.

  • Main Navigation: The top menu provides access to primary sections: Platforms, Services, Resilience, and Contact.
  • Platform Pages: Visit the Platforms section to explore our 10 integrated value-chain platforms in detail.
  • Service Information: The Services section explains our core offerings and how we work with partners.
  • Footer Links: The footer contains links to legal documents, resources, company information, and additional platforms.

Accessibility Features

We are committed to ensuring our website is accessible to all users, including individuals with disabilities. Our site complies with WCAG 2.1 AA accessibility standards.

  • Keyboard Navigation: All interactive elements can be accessed using keyboard navigation. Press Tab to move between links and buttons.
  • Screen Reader Support: Our website is fully compatible with screen readers such as NVDA, JAWS, and VoiceOver.
  • High Contrast Mode: Browser contrast settings are respected and the site remains readable in high contrast mode.
  • Text Sizing: You can increase or decrease text size using your browser's zoom function (typically Ctrl/Cmd + Plus/Minus).
  • Focus Indicators: All interactive elements display clear focus indicators when navigated using the keyboard.

Responsive Design

Our website uses responsive design technology that automatically adapts to different screen sizes and devices, including desktops, tablets, and smartphones.

  • Mobile-Friendly: The site is fully functional on mobile devices with optimized layouts and touch-friendly interface elements.
  • Auto-Scaling: Content and images automatically scale to fit your device's screen size.
  • Flexible Navigation: Mobile devices display a simplified menu that can be toggled using the hamburger icon.

Finding Information

  • Search Function: Use the search bar to quickly locate content by keyword. Search results are filtered by relevance.
  • Sitemap: A complete overview of the website structure is available via the Sitemap link in the footer.
  • Breadcrumb Navigation: Displays your current location within the site hierarchy for easy navigation.
  • Link Indicators: Icons indicate external links, downloadable files (PDF, Excel), and media content.

Browser Requirements

Our website works best with modern web browsers. For optimal performance, we recommend using the latest versions of:

  • Google Chrome (version 90 or later)
  • Firefox (version 88 or later)
  • Safari (version 14 or later)
  • Microsoft Edge (version 90 or later)

Our site requires JavaScript to be enabled for full functionality. If you have disabled JavaScript, some interactive features may not work correctly.

Troubleshooting

If you experience any issues navigating or using the website, try the following:

  • Clear Your Cache: Clear your browser's cache and cookies, then reload the page.
  • Update Your Browser: Ensure you are using the latest version of your web browser.
  • Disable Browser Extensions: Some browser extensions may interfere with website functionality. Try disabling them temporarily.
  • Try a Different Browser: Test the site in a different browser to isolate browser-specific issues.
  • Contact Support: If problems persist, please contact us at contact@yaa.group with details about the issue.

Contact & Support

If you have questions about using the website or need assistance, we're here to help. Please reach out to us at contact@yaa.group with your inquiry.
